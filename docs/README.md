# 機材レンタル管理ソフト

商品コードを軸に、機材のレンタル・返却・ステータス照会・履歴管理を行うデスクトップアプリケーションです。
専用ウィンドウで動作し、ブラウザは使用しません。

## 起動方法

**`dist/機材レンタル管理.exe` をダブルクリック** するだけで、専用ウィンドウが最大化状態で開きます。

> **終了方法**: ウィンドウの × ボタンをクリックするとアプリが終了します。

## 使用方法

### 1. 機材マスタの登録

1. 画面右上の「**機材マスタ**」リンクをクリック
2. Excelファイル（.xlsx）を用意する
   - **1行目**: ヘッダー行（読み飛ばされます）
   - **A列**: 商品コード
   - **B列**: 商品名
3. 「ファイルを選択」でExcelを選択し、「**インポート**」をクリック
4. 追加・更新・削除の件数が表示される

> **補足**: 機材を追加・削除したい場合は、Excelを編集して再インポートしてください。
> レンタル履歴がある機材はインポート時に自動で削除対象から除外されます。

### 2. レンタル登録

1. メイン画面で商品コードを入力し「**検索**」をクリック
2. ステータスが「**貸出可能**」であることを確認
3. 部署名・氏名・開始日・終了日を入力
4. 「**レンタル登録**」をクリック

### 3. 返却処理

1. メイン画面で商品コードを入力し「**検索**」をクリック
2. ステータスが「**レンタル中**」であることを確認
3. 「**返却**」ボタンをクリック

### 4. レンタル履歴の確認

商品コードで検索すると、画面下部にその機材の過去のレンタル履歴が一覧表示されます。

## 別PCへの移行手順

### コピーするもの

| 対象 | 必要 | 備考 |
|---|---|---|
| `dist/機材レンタル管理.exe` | ✅ 必須 | アプリ本体 |
| `data/rental.db` | ✅ データ引継ぎ時 | 機材マスタ＋レンタル履歴 |

### 移行先での手順

1. `機材レンタル管理.exe` と `data/` フォルダを同じディレクトリにコピー
2. `機材レンタル管理.exe` をダブルクリック

> **Pythonのインストールは不要**です。.exe 単体で動作します。

## データバックアップ

`data/rental.db` ファイル1つに全データ（機材マスタ＋レンタル履歴）が格納されています。
このファイルをコピーするだけでバックアップが完了します。

## 開発者向け情報

### セットアップ（開発環境）

```bash
# 1. プロジェクトフォルダに移動
cd 機材レンタル管理ソフト

# 2. 仮想環境を作成
python -m venv venv

# 3. 依存パッケージをインストール
.\venv\Scripts\pip.exe install -r requirements.txt
```

### 開発時の起動

```bash
.\venv\Scripts\python.exe src\app.py
```

### .exe ビルド

```bash
.\venv\Scripts\pip.exe install pyinstaller
.\venv\Scripts\pyinstaller.exe build.spec --distpath dist --workpath build_temp --clean
```

### アーキテクチャ

